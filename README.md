# interpolator

## Install and run

### Prerequisites

*   Anaconda3
    *   tensorflow or tensorflow-gpu
    *   opencv-python
    *   matplotlib
    *   joblib
    
### Running tests

#### Run all tests

Run this command:

```
python run_tests.py
```

#### Run a specific test

Run this command:

```
python -m unittest path.to.<name>_test
```

For example:

```
python -m unittest pwcnet.warp.warp_test
```
